# Unity Virtual Twin

Unity-based 3D visualization for antenna digital twin.

## Setup

1. Open Unity 2022 LTS
2. Import this project
3. Configure WebSocket connection to backend

## Scripts

- `AntennaController.cs` - Antenna parameter control
- `PatternVisualizer.cs` - 3D radiation pattern visualization
- `EnvironmentSimulator.cs` - Environment interaction
- `APIClient.cs` - Backend WebSocket communication
- `CoverageMapper.cs` - Coverage visualization

## Integration

Connect to backend via WebSocket on port 8765 (configurable).
